Deze opgave demonstreert hoe een digitale handtekening kan worden toegevoegd aan een file.

De demonstratie gebruikt de flowchart die in Figuur 6 is gegeven.

Figuur 6 Flowchart voor toevoegen van een digitale handtekening


    De hoofdlijnen zijn
  1. Open document
  2. Document
  3. Bereken een hash value
  4. Encrypt hash value
  5. Genereer handtekening
  6. Sla de handtekening op

Tijdens de demonstratie kleuren deze stappen blauw en als ze afgehandeld zijn kleuren ze groen.

Voor stap 3, 4 en 5 is invoer vereist via de rode hexagons. Die moeten dus eerst uitgevoerd worden.

Opgave 7

a) Genereer de digitale handtekening

Open vooraf een willekeurig bestand

Selecteer in menu 'Digital Signature/PKI' > 'Signature Demonstration (Signature Generation)...'.

Kies bij stap 3 de cryptografische hashfunctie SHA-1

Zie SHA-1 (wikipedia)

Laat bij stap 4 de hash code berekenen

Laat bij stap 4 de RSA key berekenen. Laat de hiervoor benodigde priemgetallen genereren

Let bij stap 5 goed op welke velden verplicht zijn


b) Verifieer de digitale handtekening

Selecteer in menu 'Digital Signature/PKI' > 'Verify Signature...'

c) Zou je SHA-1 willen gebruiken voor digitale handtekeningen?

Nee, Wikipedia (SHA-1) suggereert dat SHA-1 nodig vervangen moet worden voor digitale handtekeningen.
